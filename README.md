# Como executar o projeto

## Pré-requisitos: É preciso ter o Node instalado para usar o NPM

```bash
# clonar repositório
git clone https://github.com/nikolassco/portfolio.git
```

## Abra a pasta no VSCode

```bash
# baixar dependências
npm install
# e executar o projeto
npm start
```
